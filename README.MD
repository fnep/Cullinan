# Cullinan

Cullinan基于Tornado、SQLAlchemy编写，用于帮助项目快速搭建WEB服务

## 使用方法
    
#### 安装

将dist下whl文件放入项目目录下，并在项目目录下使用下方命令安装
        
    pip install cullinan
